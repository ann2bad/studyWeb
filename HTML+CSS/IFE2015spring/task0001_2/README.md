## 2. 给你的网页加点样式

### 2.1 任务描述

*面向零基础同学*

学习以下`CSS`是怎么运作的，然后创建一个`task0001.css`的文件，并在`task0001.html`中引入它，然后我们对`task0001.html`做一些让他变得花哨一点的事情：

- 让一级标题的文字颜色变成蓝色
- 二级标题的文字大小变成`14px`
- 段落的文字大小变成`12px`，文字颜色是黄色，带一个黑色的背景色
- 图片有一个红色的，`2px`粗的边框
